Methodology

NSI is a relative-momentum index built from five normalized sub-indicators.

NSI = 0.25·MCM + 0.20·VLA + 0.20·SMS + 0.15·DEV + 0.20·CFI
MCM · 25%

Market-Cap Momentum

Cap-weighted 7-day return across the top constituents of each narrative. Live from CoinGecko's per-category markets endpoint.

VLA · 20%

Volume / Liquidity Acceleration

24h aggregate volume ÷ market cap. A turnover spike usually precedes trend confirmation.

SMS · 20%

Social Mindshare Shift

Δ share-of-voice on Kaito + LunarCrush, 7d vs 30d. Tiered values in this build; pluggable API in roadmap.

DEV · 15%

Developer Activity

Active devs and commits via Electric Capital / Artemis. Tiered (high/med/low) — slow-moving signal.

CFI · 20%

Capital Inflows

ETF + stablecoin + bridge net inflows over 14 days. The "stickiness" check — real money on rails.

Normalization

0–100 min-max within the universe

Each sub-indicator is rescaled relative to the 21-narrative universe. NSI ≥ 65 = Accumulate, 45–65 = Hold/Watch, < 45 = Fade.

About this index

The Narrative Strength Index is a research output of the CryptoStrategy project. It is intended for research and education only — it is not investment advice. Live data is sourced from CoinGecko. Mindshare, developer activity, and flow tiers are currently set heuristically; a roadmap item is to wire them to Kaito, Electric Capital, and DefiLlama / Farside respectively.
